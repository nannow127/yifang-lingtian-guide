# 一方灵田 · 随身攻略册

基于 Vue 3、Element Plus 和攻略 Excel 数据制作的纯前端工具。支持菜谱筛选、NPC 喜好、食材来源、种植规划、收藏、已做标记与利润分析。

物价页保留攻略表中的灵植凡/良/佳售价，并支持批量补录菜品、动物产品、鱼类和商店食材价格。补录后会自动计算菜谱成本覆盖率、利润排行、作物单种期望收益与日均收益。

另外收录牧夏兑换、灯谜题库、琼珍阁解锁清单、鱼类图鉴、怪物掉落、七章主线任务与丹药效果图鉴。支持按地点查询鱼类，也可用材料名称反查掉落怪物；任务、兑换、解锁和灯谜进度保存在本地浏览器。

社区物价补充自 2022 年抢先体验攻略图，涵盖菜品、鱼类、素材与加工品，并在网页中标记为 `EA` 参考价。当前 Excel 的灵植价格优先级更高，用户自定义价格最高；社区图中被水印遮挡或未提供的价格保持为空。

## 本地运行

```bash
npm install
npm run dev
```

## 更新攻略数据

```bash
node scripts/extract-data.mjs "/path/to/攻略汇总.xlsx"
```

## 发布到 GitHub Pages

推送到 `main` 分支后，GitHub Actions 会自动构建。首次使用时在仓库 Settings -> Pages 中将 Source 设为 **GitHub Actions**。

收藏、已做记录和手动录入的菜品售价保存在浏览器 `localStorage`，不需要数据库。
